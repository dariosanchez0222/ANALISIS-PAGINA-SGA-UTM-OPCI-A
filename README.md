# ANALISIS-PAGINA-SGA-UTM-OPCI-A
Sus características principales son: Implementa un menú superior tipo barra horizontal, similar a sistemas gubernamentales y plataformas estudio, con submenús desplegables para cada macro–categoría. Las opciones se agrupan en bloques como: Información Académica, Trámites y Matrícula, Información Institucional y Configuración.
